# 제2회 상관계수 (相関係数)

## 1. 상관 (Correlation) 이란?

### 정의
두 변수 X와 Y 사이의 **선형적 관계**의 강도와 방향을 나타내는 지표

| 상관 유형 | 산점도 모양 | 의미 |
|---|---|---|
| **정의 상관 (正の相関)** | 우상향 (↗) | X가 커지면 Y도 커짐 |
| **부의 상관 (負の相関)** | 우하향 (↘) | X가 커지면 Y는 작아짐 |
| **무상관** | 흩어짐 | X와 Y 사이에 선형 관계 없음 |

---

## 2. 기술통계 vs 추측통계

| 관점 | 데이터를 보는 방식 | 상관계수 |
|---|---|---|
| **기술통계** | 데이터 자체를 요약 | 표본 상관계수 |
| **추측통계** | 데이터 = 확률변수의 실현값 | 모상관계수의 추정 |

---

## 3. 이변량 데이터의 기본

### 표기법
- N: 표본 크기 (sample size)
- $(X_1, Y_1), (X_2, Y_2), \ldots, (X_N, Y_N)$: N개의 데이터 쌍

### 산점도 (Scatterplot)
- X축: 첫 번째 변수
- Y축: 두 번째 변수
- 각 점 = 하나의 관측값 쌍

**예시**: 수험생의 수학 점수(X)와 영어 점수(Y)

---

## 4. 평균과 분산

### 표본 평균 (Sample Mean)
$$\bar{X} = \frac{1}{N}\sum_{i=1}^{N} X_i$$

$$\bar{Y} = \frac{1}{N}\sum_{i=1}^{N} Y_i$$

### 표본 분산 (Sample Variance)
$$s_X^2 = \frac{1}{N}\sum_{i=1}^{N} (X_i - \bar{X})^2$$

> 참고: N-1로 나누는 경우도 있음 (불편추정량)

### 표본 표준편차 (Sample Standard Deviation)
$$s_X = \sqrt{s_X^2}$$

---

## 5. 편차와 공분산

### 편차 (Deviation)
$$D_{X_i} = X_i - \bar{X}$$
$$D_{Y_i} = Y_i - \bar{Y}$$

> "평균으로부터 얼마나 떨어져 있는가"

### 공분산 (Covariance)
$$s_{XY} = \frac{1}{N}\sum_{i=1}^{N} (X_i - \bar{X})(Y_i - \bar{Y})$$

**공분산의 부호와 상관:**

```
        Y
        ↑
   ②    │    ①
  (-)   │   (+)
────────┼────────→ X
  (-)   │   (+)
   ③    │    ④
```

| 사분면 | 편차의 곱 | 의미 |
|:---:|:---:|---|
| ① | (+)(+) = + | 둘 다 평균보다 큼 |
| ② | (-)(+) = - | X는 작고 Y는 큼 |
| ③ | (-)(-) = + | 둘 다 평균보다 작음 |
| ④ | (+)(-) = - | X는 크고 Y는 작음 |

- **정의 상관**: ①③에 점이 많음 → 공분산 > 0
- **부의 상관**: ②④에 점이 많음 → 공분산 < 0

---

## 6. 상관계수 (Correlation Coefficient)

### 정의
$$r_{XY} = \frac{s_{XY}}{s_X \cdot s_Y} = \frac{\sum_{i=1}^{N}(X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum_{i=1}^{N}(X_i - \bar{X})^2} \cdot \sqrt{\sum_{i=1}^{N}(Y_i - \bar{Y})^2}}$$

### 성질
$$-1 \leq r_{XY} \leq 1$$

| r 값 | 의미 |
|:---:|---|
| r = 1 | 완전 정의 상관 (데이터가 우상향 직선 위) |
| r = -1 | 완전 부의 상관 (데이터가 우하향 직선 위) |
| r ≈ 0.9 | 강한 정의 상관 |
| r ≈ -0.9 | 강한 부의 상관 |
| r ≈ 0 | 거의 무상관 |

---

## 7. Z-변환 (표준화)

### 정의
$$Z_{X_i} = \frac{X_i - \bar{X}}{s_X}$$

### 성질
- Z-변환된 데이터의 평균 = 0
- Z-변환된 데이터의 분산 = 1
- **단위가 제거됨** (무차원화)

### Z-변환을 이용한 상관계수
$$r_{XY} = \frac{1}{N}\sum_{i=1}^{N} Z_{X_i} \cdot Z_{Y_i}$$

> "표준화된 편차의 곱의 평균"

---

## 8. 편차치 (偏差値)

일본 수험에서 사용하는 점수 변환법:

$$\text{편차치} = 50 + 10 \times Z$$

| 속성 | 값 |
|---|---|
| 평균 | 50 |
| 표준편차 | 10 |

## 9. 상관계수의 범위 증명 (코시-슈바르츠 부등식)

### 코시-슈바르츠 부등식 (Cauchy-Schwarz Inequality)
$$|\mathbf{a} \cdot \mathbf{b}| \leq \|\mathbf{a}\| \cdot \|\mathbf{b}\|$$

여기서:
- $\mathbf{a} \cdot \mathbf{b} = \sum_{i=1}^{N} a_i b_i$ (내적)
- $\|\mathbf{a}\| = \sqrt{\sum_{i=1}^{N} a_i^2}$ (유클리드 노름)

### 증명 아이디어
$\mathbf{a} = (X_1 - \bar{X}, \ldots, X_N - \bar{X})$로 두면:

$$|r_{XY}| = \frac{|\mathbf{a} \cdot \mathbf{b}|}{\|\mathbf{a}\| \cdot \|\mathbf{b}\|} \leq 1$$

### 등호 조건
- **r = 1**: 벡터 a와 b가 **같은 방향** → 데이터가 우상향 직선 위
- **r = -1**: 벡터 a와 b가 **반대 방향** → 데이터가 우하향 직선 위

---

## 10. 핵심 공식 요약

| 개념 | 공식 |
|---|---|
| 표본 평균 | $\bar{X} = \frac{1}{N}\sum X_i$ |
| 표본 분산 | $s_X^2 = \frac{1}{N}\sum(X_i - \bar{X})^2$ |
| 공분산 | $s_{XY} = \frac{1}{N}\sum(X_i - \bar{X})(Y_i - \bar{Y})$ |
| 상관계수 | $r_{XY} = \frac{s_{XY}}{s_X \cdot s_Y}$ |
| Z-변환 | $Z_i = \frac{X_i - \bar{X}}{s_X}$ |
